# Privacy Website Project

Simple static website project for publishing a privacy policy and `app-ads.txt`.

## Files
- `index.html` → main privacy policy page
- `styles.css` → page styling
- `app-ads.txt` → AdMob app-ads entry

## How to use
1. Upload these files to your website hosting.
2. Make sure `app-ads.txt` is accessible at:
   `https://henglihov.online/app-ads.txt`
3. Open your privacy policy page at your main domain or a path like:
   `https://henglihov.online/privacy`
